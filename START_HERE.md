## 🚨 TROUBLESHOOTING

### "npm install fails"
→ Install Node.js from nodejs.org

### "API Key Invalid"
→ Check `/src/app/utils/geminiAPI.ts` has correct key

### "Analysis fails"
→ Check browser console (F12) for errors

### "Nothing happens"
→ Make sure project name is filled

### "Port in use"
→ Run: `npx kill-port 5173`
